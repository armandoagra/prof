Scripting: introdução

Scripts são arquivos contendo código de programação que vão servir para adicionar ou alterar o comportamento de GameObjects na cena.

Em Unity, os scripts são escritos em C#. Eles permitem que você defina como os objetos do jogo interagem entre si, respondem a eventos e se comportam durante o jogo.

Criando um script

Para criar um script em Unity, siga os passos abaixo:

  1. No painel Project, clique com o botão direito na pasta onde deseja criar o script.
  2. Selecione Create > C# Script.
  3. Digite um nome para o script e pressione Enter.

Após criar o script, você pode abri-lo em um editor de código, como o Visual Studio, para começar a escrever seu código.

Estrutura básica de um script

Um script em C# para Unity geralmente tem a seguinte estrutura básica:

using UnityEngine; // Isso é um "namespace" que permite usar as funcionalidades do Unity
public class NomeDoScript : MonoBehaviour
{
    // Método chamado quando o script é iniciado
    void Start()
    {
        // Código de inicialização
    }
    // Método chamado a cada frame
    void Update()
    {
        // Código de atualização
    }
}

Os métodos Start() e Update() são dois dos métodos mais comuns usados em scripts Unity. Start() é chamado uma vez quando o script é iniciado, enquanto Update() é chamado a cada frame do jogo.

Anexando scripts a GameObjects

Para que um script tenha efeito no jogo, ele precisa ser anexado a um GameObject na cena. Para fazer isso, basta arrastar o script do painel Project para o GameObject desejado no painel Hierarchy ou Scene.

Fora isso, para que o código de um script seja executado, é necessário que ele seja "chamado":

  1. Através de eventos do Unity, como Start() e Update().
  2. Através de chamadas diretas de outros scripts.
  3. Através de colisão
  4. Através de input
Exemplo de script

Mover um objeto para frente

using UnityEngine;
public class MoveForward : MonoBehaviour
{
    public float speed = 5f; // Velocidade de movimento
    void Update()
    {
        transform.position += Vector3.forward * speed * Time.deltaTime;
    }
}

Com esse script, conseguimos um comportamento novo: um objeto que sempre anda para frente na mesma velocidade.

A partir de agora, já conseguimos criar novos objetos que usam o mesmo script mas andam em velocidades diferentes!

Vamos destrinchar esse código?

Na linha 1, importamos o namespace UnityEngine, que contém todas as classes e funções necessárias para trabalhar com Unity.

Na linha 3, definimos uma nova classe chamada MoveForward que herda de MonoBehaviour. Isso é necessário para que o script possa ser anexado a um GameObject.

Na linha 5, declaramos uma variável pública chamada speed, que define a velocidade de movimento do objeto.

Na linha 7, definimos o método Update(), que é chamado a cada frame do jogo.

Na linha 9, alteramos o valor de "position" dentro de "transform" para alterar a posição do personagem. A função Vector3.forward representa a direção para frente no espaço 3D, e multiplicamos isso pela velocidade e pelo tempo delta para garantir um movimento suave.

Agora que já vimos como é um script simples, vamos reforçar alguns conhecimentos de programação.

Conceitos básicos de programação

Antes de avançarmos para scripts mais complexos, é importante entender alguns conceitos básicos de programação:

Variáveis

Variáveis são usadas para armazenar dados. Em C#, você deve declarar o tipo da variável ao criá-la. Exemplos de tipos comuns incluem int (números inteiros), float (números decimais), string (texto) e bool (verdadeiro/falso).

int score = 0; // Variável inteira
float speed = 5.5f; // Variável decimal
string playerName = "Jogador1"; // Variável de texto
bool isGameOver = false; // Variável booleana
Vector3 direction; // Variável de vetor 3D
Repare no nome das variáveis: elas seguem o padrão "camelCase", onde a primeira palavra começa com uma letra minúscula e as próximas começam com letra maiúscula. Repare também que todas as variáveis têm nomes claros, que representam alguma coisa. Nada de chamar a variável de "a", "b", "coisa", etc.
Funções / métodos

Funções (ou métodos) são blocos de código que realizam uma tarefa específica. Elas podem receber parâmetros e retornar valores.

void AddScore(){
    score++; // Incrementa a pontuação em 1    
}

void AddScore(int points){
    score += points; // Incrementa a pontuação em "points"
}
